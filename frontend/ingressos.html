Como Ler Ingressos

Veja abaixo o passo a passo para validar ingressos manualmente:

1. Solicite o Código

Peça ao participante o código impresso ou digital que consta no ingresso.

3. Verifique a Resposta

O sistema informará se o ingresso é válido, inválido ou já foi utilizado.

4. Confirme a Entrada

Se o ingresso for válido, permita a entrada da pessoa. Se não for, oriente o participante.
